配置php到系统PATH环境变量中

建议把php可执行文件目录加入系统级别PATH环境变量中，这样带来两个好处：
1. 运行php命令时无须带全路径，简化日常操作。
2. crontab/系统脚本执行中避免出错。

1. Linux中的设置

编辑/etc/profile，在最后加入(请按php实际安装路径为准)：
export PATH=$PATH:/usr/local/php/bin
然后使用source /etc/profile使修改生效。

2. windows上的配置

右键点击桌面”此电脑“，属性，高级系统设置、环境变量、然后在系统环境变量中修改Path,添加php.exe所在的目录。
